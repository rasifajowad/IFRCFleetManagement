# IFRC Fleet Management (No GPS)

- Manual, in-city fleet scheduling with roles (Staff, Fleet Officer, Driver)
- Staff submit requests; Officer approves/assigns (override conflicts); Drivers start/end trips
- Schedule view; CSV export in Admin (Officer role)
